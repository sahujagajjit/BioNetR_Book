# BioNetR Ignite

A practical and structured learning resource for biological network analysis using R.

## 📘 Topics Covered
- Biological Networks (PPI, GRN, Functional)
- R Programming for Bioinformatics
- STRING Database Integration
- Network Analysis (Degree, Betweenness, Hubs)
- Visualization (igraph, ggraph, visNetwork)

## 🎯 Purpose
Designed for students and researchers to learn omics-based network analysis in a guided, hands-on manner.
